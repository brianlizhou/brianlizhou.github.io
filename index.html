MoodTools

MoodTools is a project I've been working on with one of my friends for over 2 years now, and it's essentially a mental health and depression aid tool.
It allows you to easily access and find support or resources, create a backup plan, and track your improvement over time (mood monitoring).
The inital version is out right now, but we're planning on publishing a big new release in a few months; I've attached a few sneak peek screenshots below!
It currently is available on Google Play / Apple Store and has over 600,000 users!

Docker/Shell Project: Open Source Sakai

I worked with a friend on the Sakai open source project to create a dockerfile that significantly simplified the initial installation procedure.
Normally, you have to manually set up and install hundreds of files, while establishing tons of configuration files, but with our docker file/container you can just run it once and let it do all the work for you (without bugs).
Also continuously integrated with github so it updates/rebuilds on change.
